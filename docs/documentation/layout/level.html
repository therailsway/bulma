层结构如下：

  • level：主容器
    • level-left 用于左侧
    • level-right 用于右侧
      • level-item 每一个独立的元素

在一个 level-item 里，您可以插入任何您想要的：标题、按钮、输入框或者就是一段文字。不管您放入什么元素到 Bulma 的 level 里，它们都会垂直居中对齐。

{% include elements/snippet.html content=nav_example horizontal=true more=true %} {% include elements/anchor.html name="居中的层" %}
如果您想要一个居中的层，您可以使用任意多个 level-item，只要它们是 level 容器的 直接 子级即可。
{% include elements/snippet.html content=nav_centered_example horizontal=true more=true %} {% include elements/snippet.html content=nav_centered_bis_example horizontal=true more=true %} {% include elements/anchor.html name="移动设备上的层" %}
默认情况下，由于空间原因，在移动设备上层是垂直堆叠的。如果您想要层在移动设备上也水平布局，在 level 容器上添加 is-mobile 修饰符即可。
