{% highlight html %}{{ tile_default }}{% endhighlight %} {% include elements/anchor.html name="修饰符" %}

tile 元素有 16 个修饰符：

  • 3 个上下文相关的修饰符
    • is-ancestor
    • is-parent
    • is-child
  • 1 个方向相关修饰符
    • is-vertical
  • 12 个横向型号修饰符
    • from is-1
    • to is-12
{% include elements/anchor.html name="它如何运作：嵌套" %}

万物皆贴片！创建一个贴片网格，您只需要嵌套 tile 元素。

用一个封装所以其他贴片的 ancestor 贴片开始：

{% highlight html %}{{ tile_ancestor }}{% endhighlight %}

添加水平分布的贴片元素：

{% highlight html %}{{ tile_horizontal }}{% endhighlight %}

您可以根据 12 列网格重新规划任何贴片。
例如，is-4 会占用水平空间的 1/3 位置：

{% highlight html %}{{ tile_resize }}{% endhighlight %}

如果您想要垂直方向堆叠贴片，在其上级贴片上添加 is-vertical 修饰符：

{% highlight html %}{{ tile_vertical }}{% endhighlight %}

如果您想为某一个贴片添加内容，只需：

  • 如愿添加任意类，例如 box
  • 在贴片上添加 is-child 修饰符
  • 在上级贴片上添加 is-parent 修饰符
