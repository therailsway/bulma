Bulma 的高度可定制化要感谢遍布 28 个文件的 419 个 Sass 变量。

这些变量分 4 个层面：

  • 初始变量：带有字面值的全局变量
  • 衍生变量：带有值（参考其它变量或者计算而得）的全局变量
  • 通用变量：不带有 CSS 类的 HTML 元素
  • 元素（组件）变量：在 Bulma 元素（组件）内使用的变量

由于这些变量带有 !default 标记，所以不管是在导入前还是导入后，它们都可以分配新值。

定制 Bulma，您需要：

  • 安装（或者下载）Bulma
  • 有一个可工作的 Sass 安装
  • 创建您自己的 .scss 或者 .sass 文件

这可以通过以下任何一种方式实现：

  • node-sass
  • the Sass CLI
  • webpack
