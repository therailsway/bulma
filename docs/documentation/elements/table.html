创建 Bulma 表格只需在具有如下结构的 HTML 元素 <table> 上添加一个 CSS 类 table 即可：

  • <table class="table"> 做为主 容器
    • thead 为可选的表格顶部
    • tfoot 为可选的表格底部
    • tbody 为表格主内容区
      • tr 为表格的每一行
        • th 是表格的单元格标题
        • td 是表格的单元格
