Bulma 是一个 CSS 库。这也就是说，它能帮您美化您的 HTML 代码。
使用 Bulma，您可以使用预编译的 .css 文件，也可以安装能够自由定制的 .sass 文件。

获取 Bulma CSS 文件

一个包含所有 Bulma 代码的单独的 .css 文件

推荐 选项 1. 使用一个 CDN

您可以直接从 jsDelivr 导入 CSS 文件：
