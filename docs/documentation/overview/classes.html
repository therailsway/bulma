Bulma 是一个 CSS 框架，也就是说最终会简化为一个单一的 .css 文件：
https://github.com/jgthms/bulma/blob/master/css/bulma.css

因为 Bulma 只包括 CSS 类，所以您的 HTML 代码不影响页面的样式。这就是为什么 .input 会做为一个类存在的原因，这样您就可以选择美化哪个 <input type="text"> 元素了。

Bulma only styles generic tags directly twice:

  • generic.sass 定义页面基本样式
  • .content 类用于任何文本内容，如同 WYSIWYG 一样
