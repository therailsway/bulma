Bulma 的 breadcrumb 是一个简单的导航组件——只需要一个 breadcrumb 容器和一个 ul 列表即可。分隔符会自动由 li 标签的 ::before 伪元素自动生成。

您可以使用 is-active 修饰符在当前页的 li 标签上做标记。这会使导航的内部链接失效。

除了默认的对齐方式，还可以在 breadcrumb 容器上使用 is-centered 和 is-right 修饰符。
