panel 是容器：

  • panel-heading 是第一个子元素
  • panel-tabs 用于导航
  • panel-block 可包含其他元素：
    • control
    • input
    • button
    • panel-icon

panel-block 可以是锚点标签 <a> 或者带有内置选择框的标签 <label>。
