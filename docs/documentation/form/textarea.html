Bulma 为 textarea 元素定制了不同的状态。每一种状态可用伪类和 CSS 类设置：

  • :hover 和 is-hovered
  • :focus 和 is-focused
  • :active 和 is-active

这使得您可以不需要触发而获得特定状态。
