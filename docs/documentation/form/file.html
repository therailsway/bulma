file 元素是封装了 <input type="file"> 的简单的互动标签。它包含几个子元素：

  • file 是主容器
    • file-label 是实际互动和可点击的元素
      • file-input 是原生的文件输入框，基于美化目的已隐藏
      • file-cta 上传调用操作
        • file-icon 是可选的上传图标
        • file-label 显示文字“选择一个文件……”
      • file-name 一个包含选择的文件名字的容器
